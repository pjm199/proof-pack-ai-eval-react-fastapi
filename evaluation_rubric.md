# Mini Evaluation Rubric (30 pts)
1) Correctness (0–10) — All tests pass, edges handled.  
2) Code Quality (0–6) — Clarity, idioms, formatting.  
3) Tests (0–6) — Unit tests, negative paths, determinism.  
4) Efficiency (0–4) — Appropriate complexity & structures.  
5) Docs & DX (0–4) — README, commands, examples.
Pass bands: 24–30 Excellent · 18–23 Good · 12–17 Needs work · <12 Insufficient.
